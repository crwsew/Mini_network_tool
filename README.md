#  Mini Network Tool

A simple Python CLI tool to practice networking concepts:
- HTTP Requests
- DNS Lookup
- Ping Test

## Features

- Send HTTP GET requests
- Lookup IP address of a domain
- Ping a domain or IP
- Logging all activities

## Installation

git clone https://github.com/crwsew/Mini_network_tool.git
cd Mini_network_tool
pip install -r requirements.txt


## Concepts Covered

- Client / Server
- HTTP Request & Response
- DNS
- TCP/IP
- Basic Networking
